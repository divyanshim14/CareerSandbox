# CareerSandbox

Try careers before marrying them.

## Problem
Students choose careers with limited exposure and often discover misalignment too late.

## Solution
CareerSandbox provides:
- Personalized career recommendations
- Interactive career simulations
- Adaptive difficulty
- Progress dashboard

## Features
- Career profiling
- Recommendation engine
- 7 career simulations
- Analytics dashboard
- Responsive design

## Tech Stack
Frontend:
- HTML
- CSS
- JavaScript

Backend:
- Node.js
- Express

Database:
- SQLite
